Install Instructions:
1. Unzip this folder (Polyvox)
2. Open Chrome to the url "chrome://extensions"
3. Turn on Developer Mode in the top right of the page
4. Click the "Load unpacked" button in the top left of the page
5. Select this folder (Polyvox)
Polyvox is now loaded on to your browser! :>

Use Instructions:
1. Navigate to a valid website (ex. a news article, a blog post)
	NOTE: Polyvox will NOT work on websites where HTML extraction is not allowed (the Chrome extension page, the default New Tab page)
2. Click on the green globe icon in the top right
Polyvox is now open to your active page! You can customize what language market your results come from.
